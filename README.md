# Cubism4WebGL_Samples

Qiitaの「[Cubism 4 SDK for Web でモデルを表示する](https://qiita.com/tatsuteb/items/2c00855968acd9f4b7df)」用サンプルコード

## ビルド方法

1. `git submodule update --init` # SDKをチェックアウト
2. `npm run copy` # SDKからモデルをdistにコピー
3. `npm start`
